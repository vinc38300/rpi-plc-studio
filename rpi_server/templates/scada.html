📖 Comment configurer

Étape 1 — Créer le bot

Ouvrir Telegram et chercher @BotFather.
Taper /newbot, choisir un nom.
Copier le token fourni.

Étape 2 — Trouver votre Chat ID

Démarrer une conversation avec votre bot, puis visiter :
https://api.telegram.org/bot<TOKEN>/getUpdates
Votre chat_id apparaît dans la réponse JSON.

Étape 3 — Tester

Sauvegarder la config, cliquer "Envoyer test".
Vous devez recevoir un message Telegram.

🌐 Accès à distance (Tailscale)

La solution recommandée : Tailscale

Tailscale crée un VPN mesh privé. Votre RPi obtient une IP fixe accessible depuis n'importe où, sans ouvrir de port sur votre box.

Installation sur le RPi

curl -fsSL https://tailscale.com/install.sh | sh sudo tailscale up # → Scanner le QR code avec l'app Tailscale # Votre RPi reçoit une IP 100.x.x.x

Accès depuis votre téléphone

1. Installer Tailscale sur votre téléphone
2. Se connecter avec le même compte
3. Accéder à http://100.x.x.x:5000/scada

✓ Avantages Tailscale

• Gratuit jusqu'à 100 appareils
• Chiffrement WireGuard (militaire)
• Aucun port à ouvrir sur la box
• Fonctionne derrière CGNAT
• iOS / Android / Windows / Linux
